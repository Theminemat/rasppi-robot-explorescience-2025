# Raspberry Pi Robot — Explore Science 2025

> **Disclaimer:** This was a small robotics project built in just **5 days**
> for a regional German STEM competition called *Explore Science* (Bremen, 2025).
> This doesn't exactly follow any proper code standards — expect rough edges, minimal documentation,
> and quick-and-dirty solutions made under time pressure.
> Shared here for archival purposes only (and as a warning to future me).

## What is this?

A Raspberry Pi-powered robot developed for the **Explore Science** competition
in Bremen, Germany — a hands-on science and technology event for young students
focused on robotics, sensor integration, and autonomous systems.

The robot was supposed to collect 100 ping pong balls on a 2×2m field
as fast as possible.

## Result

We actually placed in the competition and won a prize. Yeah, we were surprised too
looking at this project.

## Stack

- **Hardware:** Raspberry Pi
- **Language:** Python / C++ (embedded)
- **Interface:** Web UI (see `/web_ui`)

## Context

Built as part of a 5-day sprint. The goal was a working robot on the
competition floor, not clean, maintainable software.
Most parts are undocumented because getting it to work was the priority.

## Status

This repo is **not actively maintained**. It exists as a snapshot of the
project as it was at the competition.
